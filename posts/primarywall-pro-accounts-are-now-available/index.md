Good news everyone! Our amazingly popular service [PrimaryWall](http://primarywall.com) now has pro accounts available so you can extend functionality and provide a personalized experience! So what’s new?

Schools can now register their own subdomain for PrimaryWall for example http://stjohns.primarywall.com.

## Change font

We have provided 16 beautiful fonts, when you change a font it instantly changes for everyone else too!

[![](images/font.png "font")](http://blog.primaryt.co.uk/files/2011/05/font.png)

## Password protect walls

By default pro accounts are only accessible by Wall owners, the owner can then set a password so pupils can access a specific wall.

[![](images/security.png "security")](http://blog.primaryt.co.uk/files/2011/05/security.png)

## Change background

[Safe Search](http://primaryschoolict.com) have provided us with 8 topical backgrounds that you can apply, there is an also a blank background so you can get more a focus on the notes. When you change the background it automatically updates on everyone else’s wall.

[![](images/background.png "background")](http://blog.primaryt.co.uk/files/2011/05/background.png)

## Easy wall embed

Easy embed of walls means you can keep your kids in your safe blog/learning platform environment while still using PrimaryWall. This works with an iframe embed code which is supported by all good platforms. [Here are some instructions on how to embed PrimaryWall on PrimaryBlogger](http://blog.primaryt.co.uk/2011/05/10/how-to-embed-primarywall-in-primaryblogger/)

[![](images/embed.png "embed")](http://blog.primaryt.co.uk/files/2011/05/embed.png)
