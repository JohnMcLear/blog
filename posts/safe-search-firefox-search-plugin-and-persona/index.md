Today we are releasing a [Firefox Safe Search plugin](http://mycroft.mozdev.org/search-engines.html?name=safe+search+Primary+Technology&opensearch=yes). The [Safe Search plugin](http://mycroft.mozdev.org/search-engines.html?name=safe+search+Primary+Technology&opensearch=yes) allows you to easily search using safe search from the Firefox toolbar.

Installation:

1. Visit the [firefox safe search plugin search results page](http://blog.primaryt.co.uk/2011/04/29/safe-search-firefox-search-plugin-and-persona/) when using firefox
2. Click on Safe Search from Primary Technology
3. Click Add

Also here is our [Safe Search firefox persona](http://www.getpersonas.com/en-US/persona/397533) – this will make your Firefox a bit prettier! ![:)](http://blog.primaryt.co.uk/wp-includes/images/smilies/icon_smile.gif)
